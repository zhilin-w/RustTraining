## 要点

- 使用 `tokio::net::TcpListener` 接受连接
- 为每个连接分配任务并使用 `JoinSet` 或 `TaskTracker` 跟踪
- 使用 `mpsc` 或 `broadcast` 实现房间广播与消息路由
- 支持优雅关机：使用 `watch` 通道通知所有连接
- 在测试中使用 `time::pause()` 与 `tokio::test` 提供可控时间
